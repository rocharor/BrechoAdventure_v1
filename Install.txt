2º - Baixar dependencias NPM executar na Public (npm install);
3º - Ajustar configuração do banco, copiar arquivo "mysql.ini.sample" para "mysql.ini" e inserir os dados do banco;

5º - Apagar arquivos desnecessários que NPM baixa, deixar apenas os que serão incluidos;
